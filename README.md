## О проекте

В конце XX века мировая экономика столкнулась с «новой глобализацией», которая сопровождалась активным ростом международной торговли, инвестиций и миграции, а также трансформацией глобальных цепочек создания стоимости. Одним из ключевых событий стало усиление роли Китая на международной арене — результат активного наращивания национального производства и вступления в ВТО. Китайские товары заняли значительную долю импорта в развитые страны, непосредственно влияя на объём внутреннего производства. В научной среде это явление получило название «китайского шока».

Данный репозиторий содержит воспроизводимый эмпирический анализ «китайского шока» по методологии Отора, Дорна и Хансона (2013). Анализ проводится для двух географических контекстов:

- **США** — на уровне коммьюнити-зон (Commuting Zones)
- **Европа** — на уровне регионов NUTS-2 и NUTS-3 (Германия, Франция, Италия, Нидерланды, Испания)

Основной метод оценки — Bartik shift-share IPW с инструментальной переменной (IV-2SLS), где в качестве инструмента используется импорт из Китая в другие развитые страны (Австралия, Финляндия, Япония, Норвегия, Новая Зеландия, Швеция).

## Запуск анализа

### США

1. Откройте терминал или RStudio и установите рабочую директорию:
   ```r
   setwd("путь/к/репозиторию/USA")
   source("USA.R")
   ```
2. Скрипт автоматически загружает данные из `data/`, выполняет регрессии и сохраняет результаты в `data_out/` и `figures/`.

### Европа

1. Установите рабочую директорию:
   ```r
   setwd("путь/к/репозиторию/EUROPE")
   source("EUROPE.R")
   ```
2. При запуске откроется диалог выбора файла — выберите нужный `.xlsx` из папки `data/NUTS 2/` или `data/NUTS 3/`.
3. Задайте значения `COUNTRY` и `COUNTRY_CODE` в начале скрипта (например, `"France"` и `"fr"`).
4. Результаты сохраняются в `EUROPE/figures/`.

## Описание данных

### США (`USA/data/`)

| Путь | Содержимое |
|---|---|
| `raw/period_1999_2011/` | Занятость по отраслям NAICS4 и коммьюнити-зонам за 1999 и 2011 гг. |

| `processed/delta_import_IV_*.csv` | Изменения импорта из Китая в другие страны по отраслям (инструмент IV) |

| Папка | Содержимое |
|---|---|

## Методология

Анализ следует методологии Отора, Дорна и Хансона (2013):

1. **IPW** (Import Penetration per Worker) — Bartik shift-share мера импортного воздействия
2. **OLS** — базовая оценка с кластеризованными стандартными ошибками на уровне региона/CZ
3. **IV-2SLS** — двухшаговый МНК; инструмент — изменения импорта из Китая в другие страны (other-country imports), удовлетворяющие условию исключения
4. **Тест Хаусмана** — проверка эндогенности переменной IPW

Анализируемые периоды:

| Модуль | Периоды |
|---|---|
| США | 1999–2011 (рост китайского импорта), 2012–2019 («Шок 2.0») |
| Европа | 2000–2007, 2008–2011, 2012–2019 |
